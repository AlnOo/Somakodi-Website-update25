What You will Learn

Unit 1: Cybersecurity Foundations

In this module, students will explore the core principles of cybersecurity, including confidentiality, integrity, and availability. You'll learn about common threats, types of attacks, and the importance of securing systems and data. This module sets the groundwork for understanding how cybersecurity works and why it’s essential in today’s digital world.

Unit 2: Networking & Security Protocols

This module introduces students to the fundamentals of computer networks and how data moves across them. You’ll learn about IP addressing, DNS, firewalls, and how devices communicate securely using protocols like HTTPS, SSL/TLS, and VPNs. Understanding how networks operate—and how to protect them—is key to identifying and mitigating cyber threats.

Unit 3: System & OS Security

In this module, students will learn how to secure computer systems and operating systems against common threats. You’ll explore how attackers exploit system vulnerabilities and how to defend against them using tools like antivirus software, system updates, access controls, and secure configurations. The focus is on protecting both Windows and Linux environments, giving you practical skills to secure endpoints and servers.

Unit 4: Ethical Hacking & Penetration Testing

This module introduces students to the mindset and techniques of ethical hackers—security professionals who test systems to find and fix vulnerabilities before malicious hackers can exploit them. You’ll learn the basics of penetration testing, including reconnaissance, scanning, exploitation, and reporting. Using real-world tools like Kali Linux, Nmap, and Metasploit, you'll gain hands-on experience simulating attacks in a controlled environment to better understand how to defend against them.

Unit 5: Cloud & Application Security

In this module, students will learn how to secure cloud-based systems and web applications from modern cyber threats. You’ll explore common vulnerabilities like SQL injection and cross-site scripting (XSS), and understand how to secure applications throughout the development lifecycle. The module also covers key concepts in cloud security, including identity and access management, data protection, and compliance across platforms like AWS and Microsoft Azure.

Unit 6: Risk Management & Compliance

This module focuses on identifying, assessing, and managing cybersecurity risks within an organization. Students will learn how to evaluate potential threats, implement mitigation strategies, and ensure compliance with global security standards and regulations such as GDPR, ISO 27001, and Kenya’s Data Protection Act. You'll also explore the role of policies, audits, and governance in maintaining a strong security posture.
